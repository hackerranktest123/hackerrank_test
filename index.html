Longest Path in DAG

You are given a Directed Acyclic Graph (DAG) with N nodes and M edges. Each edge has a weight. You are also given an integer K.

Task

Find the maximum path sum from node 1 to node N using at most K edges.

Input Format

N = 5
M = 5
edges = [
  [1, 2, 3],
  [1, 3, 2],
  [2, 4, 4],
  [3, 4, -1],
  [4, 5, 6]
]
K = 3

Output Format

Print the maximum path sum from node 1 to N using ≤ K edges. If impossible, print "IMPOSSIBLE".

Sample Output

13

Explanation

The longest path is 1 → 2 → 4 → 5 with sum = 3 + 4 + 6 = 13 using 3 edges (≤ K).
